# Resume Match Assistant

AI-powered mobile application designed to compare resumes against job descriptions and provide actionable recommendations for improving candidate-job fit.

## Overview

Resume Match Assistant was created as an MVP focused on helping job seekers evaluate how well their resume aligns with a target role.

The application analyzes resume content and job requirements, identifies gaps, and generates practical recommendations to improve application quality.

## Problem

Many candidates struggle to understand why their resumes do not match specific vacancies. Manual comparison is time-consuming and often subjective.

## Solution

Resume Match Assistant automates the comparison process, highlights alignment gaps, and generates personalized recommendations to improve application effectiveness.

## Key Features

* Resume analysis
* Job description analysis
* AI-powered match scoring
* Skills gap identification
* Resume improvement recommendations
* Personalized cover letter generation
* Application optimization support
* Structured fit assessment

## Tools Used

* Google AI Studio
* Gemini AI
* Kotlin
* Android Studio
* Prompt Engineering
* AI-Assisted Product Development

## Outcome

Developed as an MVP demonstrating how AI can support job seekers in evaluating resume alignment, improving application materials, and generating tailored cover letters for target roles.
